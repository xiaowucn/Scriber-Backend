#+TITLE: The Customizable Front-End of 'Remarkable'

** Concepts

- path: a way to select one or more json nodes, like xpath to XML. Here we follow the syntax of `jq`'s filter.


** Schema

#+BEGIN_SRC javascript
  {
      "extensible": {
          "path:.preattributes": {
              "mode": "subtext(path:.text)"
          }
      },    // 用户可以自己添加的东西
      "label": {
          "time": "path:.times[]",
          "prea": "path:[.preattributes[]]",
          "attr": "path:.attributes[]",
          "value": "path:.values[]"
      },
      "ui": {
          "material": {
              "path:/source_text": {
                  "description": "原句"
              },
              "path:/text": {
                  "description": "展开后文本"
              }
          },
          "label": {
              "time": {
                  "description": "时间",
                  "show": "path:.value"
              },
              "prea": {
                  "description": "定语",
                  "show": "path:.value"
              },
              "attr": {
                  "description": "属性",
                  "show": "path:.value"
              },
              "value": {
                  "description": "值",
                  "show": "path:.value"
              }
          }
      }

  }
#+END_SRC
