# 向标注系统推送数据使用的格式

向标注系统推送数据时, 一个批次可以推送多条数据, 因此推送数据格式是一个数组.

```js
[
    {
        'doc_info': document_info,      // 文档信息,可以为空. 一个文档可以包含多个题目. 对于没有文档概念的系统来说, 一个文档就是一个题目, 此时该字段置空.
        'sid1-1': {                     // 这个对因果来说,应该是原来推送格式中的key, 页面上叫sid
            'question': question_obj,   // 题目内容
            'health': 2,                // 答题人数
            'answer': answer_obj        // 系统推荐的标注答案,可以为空
        },
        'sid1-2': {
            'question': question_obj,   // 题目内容
            'health': 2,                // 答题人数
            'answer': answer_obj
        },
        ...                             // 同文当中的其他题目
    },
    ...                                 // 其他文档数据
]
```
